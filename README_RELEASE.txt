Zapret Kmestu v0.1 beta

Zapret Kmestu — удобная Windows-оболочка для запуска, настройки и обслуживания zapret без ручной работы с bat-файлами.

Приложение помогает:
- установить zapret;
- выбрать профиль обхода;
- включать и выключать обход из понятного интерфейса;
- запускать автоподбор профиля;
- проверять состояние подключения;
- управлять базовыми настройками без командной строки.


КАК ЗАПУСТИТЬ

1. Распакуйте архив в любую удобную папку.
2. Запустите ZapretKmestu.exe.
3. Windows попросит права администратора — нажмите “Да”.

Права администратора нужны, потому что приложение работает с системными службами Windows и файлами zapret.


ЕСЛИ WINDOWS SMARTSCREEN ПОКАЖЕТ ПРЕДУПРЕЖДЕНИЕ

Это ожидаемо для beta-версии без цифровой подписи.

Нажмите:
1. “Подробнее”
2. “Выполнить в любом случае”

Также антивирус может реагировать на WinDivert. WinDivert используется zapret для работы с сетевым трафиком и сам по себе не означает, что программа является вирусом.


КАК ЗАПУСКАТЬ ВСЕГДА ОТ ИМЕНИ АДМИНИСТРАТОРА

При обычном запуске ничего дополнительно делать не нужно: Zapret Kmestu сам запрашивает права администратора через Windows UAC.

Если вы хотите, чтобы приложение запускалось вместе с Windows:

1. Откройте Zapret Kmestu.
2. Перейдите в “Настройки”.
3. Включите “Запускать Zapret Kmestu вместе с Windows”.

После этого Windows будет запускать приложение через стандартный Планировщик задач с повышенными правами.


КАК ОБНОВИТЬ ZAPRET KMESTU

1. Закройте старую версию приложения.
2. Распакуйте новую версию в новую папку или замените старый ZapretKmestu.exe.
3. Запустите новую версию.

Настройки приложения хранятся в профиле пользователя Windows, поэтому обычно сохраняются после обновления.


КАК ПОЛНОСТЬЮ УДАЛИТЬ ZAPRET KMESTU

Если приложение больше не нужно, удалите его файлы из системы.

1. Выключите обход, если он включён.
2. Закройте Zapret Kmestu.
3. Если приложение свернулось в трей, нажмите по значку Zapret Kmestu в трее и выберите “Выход”.
4. Удалите папку, где лежит ZapretKmestu.exe.

Чтобы удалить настройки, логи и сохранённые данные Zapret Kmestu, также удалите папки:

- %AppData%\Zapret Kmestu
- %LocalAppData%\Zapret Kmestu

Если вы включали автозапуск, но уже удалили приложение, откройте Планировщик заданий Windows и удалите задачу Zapret Kmestu, если она там осталась.


ЧТО ОСТАНЕТСЯ ПОСЛЕ УДАЛЕНИЯ ПРИЛОЖЕНИЯ

Удаление Zapret Kmestu удаляет только оболочку приложения.

Файлы установленного zapret могут остаться отдельно в:

C:\ProgramData\Zapret Kmestu

Если вы хотите полностью очистить систему, после выключения обхода можно удалить и эту папку.


ВАЖНО ПРО BETA-ВЕРСИЮ

Zapret Kmestu находится в стадии beta-тестирования. Несмотря на то, что приложение уже подходит для повседневного использования, в отдельных сценариях могут встречаться ошибки и неточности интерфейса.

Проект активно развивается и будет поддерживаться. Ваши отзывы помогают улучшать стабильность, удобство и качество работы приложения.

Если один профиль не помогает, используйте автоподбор или попробуйте другой профиль.
