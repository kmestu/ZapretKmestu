# Zapret Kmestu v0.1 beta

Неофициальная графическая оболочка (GUI) для Windows, предназначенная для удобного управления проектом [zapret-discord-youtube](https://github.com/Flowseal/zapret-discord-youtube) от Flowseal.

> **Важное предупреждение:** Zapret Kmestu — это **неофициальная** графическая оболочка. Данный проект не является оригинальным проектом `zapret` и никак не связан с его авторами (Flowseal или bol-van). Движок `zapret` и все связанные сторонние компоненты принадлежат их соответствующим авторам.

## 🚀 Особенности
*   Установка и удаление движка `zapret` прямо из интерфейса.
*   Удобный запуск и остановка обхода блокировок.
*   Выбор профилей обхода.
*   Автоматический подбор профиля и история результатов проверок (интегрировано в приложение).
*   Встроенная диагностика (HTTPS-отклик и Доступность).
*   Пользовательский интерфейс для игровых фильтров (если включено).
*   Поддержка светлой и темной тем оформления.
*   Поведение сворачивания в системный трей.

## ⚙️ Системные требования
*   ОС: Windows 10 или Windows 11.
*   **Права администратора** (необходимы для управления сетевыми службами).
*   Активное подключение к интернету для загрузки движка `zapret` и обновлений.

## 📖 Как использовать
1.  Запустите приложение **от имени администратора**.
2.  Нажмите кнопку для установки (скачивания) движка `zapret`.
3.  Выберите подходящий профиль обхода или воспользуйтесь рекомендациями приложения.
4.  Нажмите кнопку для запуска обхода блокировок.

## 🛡️ Безопасность и предупреждения
*   Данное приложение **не заменяет** оригинальный проект `zapret`, а лишь предоставляет удобный интерфейс.
*   В целях безопасности скачивайте релизы Zapret Kmestu **только** из официального репозитория на GitHub (после его публикации).
