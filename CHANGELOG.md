# История изменений (Changelog)

## [v0.1 beta] - Начальный публичный релиз
Первая бета-версия приложения Zapret Kmestu.

### Добавлено
*   Удобный графический интерфейс (GUI) для управления обходом блокировок на Windows.
*   Функционал автоматической установки и удаления актуальной версии движка `zapret-discord-youtube`.
*   Управление службами обхода: старт, стоп.
*   Интерфейс для выбора готовых профилей.
*   Автоматический подбор оптимального профиля с сохранением истории результатов (встроено в приложение).
*   Вкладка диагностики с проверкой HTTPS-отклика и доступности ресурсов.
*   Настройки поведения: автозапуск, сворачивание в трей, светлая/темная тема.
*   В интерфейс настроек добавлен визуальный индикатор будущей функции "Автообновление Zapret Kmestu" (в разработке, функционал автообновления пока не реализован).

### Известные ограничения
*   Приложение находится в стадии бета-тестирования, возможны непредвиденные ошибки взаимодействия со службами ОС.
*   Автоматическое обновление самого графического приложения Zapret Kmestu пока не реализовано (добавлен только интерфейс-заглушка).
